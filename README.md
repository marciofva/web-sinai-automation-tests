# Cypress 
---

E2E testing project using Cypress


### Install Cypress

```
npm install cypress --save-dev
```


### Install all dependencies

```
npm install
```


### Running the tests

- Open cypress dashboard:
```
npm run cy:open
```

- Chrome browser:
```
npm run cy:runChromeProd
```

- Firefox browser:
```
npm run cy:runFirefoxProd
```

- Headless:
```
npm run cy:runChromeProdHeadless
```
